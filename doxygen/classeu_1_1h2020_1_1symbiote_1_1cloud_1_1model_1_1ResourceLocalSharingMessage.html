Detailed Description

Class used to send sharing and unsharing information by RabbitMQ. It's created at Registration Handler side and then passed to the ${rh.exchange.name} exchange. When it's passed by ${rh.resource.local.shared} key, it contains shared resources. When using ${rh.resource.local.unshared} key, it contains unshared resources
